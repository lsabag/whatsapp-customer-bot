# בוט שירות לקוחות לוואטסאפ (דמו) — מבוסס Claude

בוט שמקבל הודעות וואטסאפ, עונה בעזרת Claude על סמך מידע על העסק, וזוכר שיחות קודמות פר-לקוח.

> ⚠️ **לדמו בלבד.** `whatsapp-web.js` אינו רשמי ועובד דרך WhatsApp Web. הוא נגד תנאי השימוש של וואטסאפ ויש סיכון לחסימת מספר — השתמש/י במספר ייעודי/זניח, לא במספר העסקי. לפרודקשן יש לעבור ל-WhatsApp Cloud API / Twilio.

## דרישות מוקדמות
- **Node.js 18+** (מותקן: גרסה 24 ✅)
- **מפתח API של Anthropic** (יש לך ✅)
- **מספר וואטסאפ ייעודי** לסריקת QR (לא העסקי)

## התקנה

```bash
npm install
```

## הגדרה

1. העתק/י את `.env.example` ל-`.env`:
   ```bash
   cp .env.example .env
   ```
2. ערוך/י את `.env` והכנס/י את מפתח ה-API שלך (`ANTHROPIC_API_KEY`).

## הרצה

```bash
npm start
```

בהרצה הראשונה יוצג בטרמינל **קוד QR**. סרוק/י אותו בוואטסאפ:
**הגדרות → מכשירים מקושרים → קישור מכשיר**.

אחרי החיבור יופיע `✅ הבוט מחובר ופעיל!` — שלח/י הודעה למספר המחובר והבוט יענה.
ה-session נשמר, כך שבהרצות הבאות לא צריך לסרוק שוב.

## איך זה בנוי

```
src/
  index.js      ← חיבור whatsapp-web.js, QR, מאזין להודעות
  bot.js        ← מחבר זיכרון + מידע עסקי + Claude (ה-system prompt כאן)
  claude.js     ← קריאה ל-Anthropic API
  memory.js     ← זיכרון מתמשך פר-לקוח (קבצי JSON תחת data/)
  knowledge.js  ← טוען את כל מסמכי knowledge/*.md
knowledge/      ← המידע שהבוט עונה לפיו (דמה — החלף בתוכן שלך)
  business.md, products.md, policies.md, faq.md
data/           ← נוצר אוטומטית: session של וואטסאפ + זיכרון שיחות
```

## התאמות נפוצות

- **לעדכן את המידע על העסק:** ערוך/י את הקבצים ב-`knowledge/` (או הוסף/י קבצי `.md` חדשים). נטענים אוטומטית בעלייה.
- **לשנות אישיות/הנחיות לבוט:** ערוך/י את `SYSTEM_PROMPT` בקובץ `src/bot.js`.
- **לשנות מודל (מהירות/עלות):** שנה/י `MODEL` ב-`.env`. אפשרויות: `claude-opus-4-8` (חכם), `claude-sonnet-4-6` (מאוזן), `claude-haiku-4-5` (מהיר/זול).
- **אורך זיכרון לכל לקוח:** `MAX_MESSAGES` בקובץ `src/memory.js` (ברירת מחדל 40 הודעות אחרונות).

## מעבר לפרודקשן (בהמשך)

המוח של הבוט (`bot.js`, `claude.js`, `memory.js`, `knowledge.js`) נשאר זהה.
רק שכבת הערוץ ב-`index.js` (whatsapp-web.js) מוחלפת ב-Twilio / WhatsApp Cloud API.
